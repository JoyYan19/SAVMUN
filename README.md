# SAVMUN
Savio Model United Nations

## Local preview
1. Open `index.html` in your browser.
2. The page contains:
   - Home hero/navigation
   - Schedule section
   - Committees section
   - Position paper submission form (front-end demo)
   - Delegates rules/FAQ section

## Using with Wix
1. Add an HTML iframe or custom element on your Wix site.
2. Paste contents of `index.html`.
3. Swap placeholder file actions with your Wix media/file endpoints.
4. Add backend handling if you need real PDF uploads (Wix Velo recommended).
